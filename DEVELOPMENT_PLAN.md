## **個人網站重構開發計畫書 (DEVELOPMENT_PLAN.md)**

### **專案目標**
將現有的個人網站重構成一個基於「CHRONOS STUDIO」風格的現代化、高性能、視覺驅動的網站。採用前後端分離架構。

### **核心技術棧**
*   **前端**: Next.js, React, TypeScript, GSAP
*   **後端**: Python (Flask) 作為 REST API 伺服器
